# 啥时候吃饭

- [项目文档](https://docs.pguide.cloud/project-docs/when2eat/)

哈哈，在国内想要高峰期吃上一口热饭可太不容易了，特别是每年新生入学军训期间，根本抢不过。啥时候吃饭(when2eat) 通过计算机视觉技术对画面中的人进行计数，配合机器学习算法，给出最合理的吃饭时间段，让你合理避开高峰期。
